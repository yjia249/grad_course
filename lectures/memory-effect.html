How solids store histories of shear, amplitude, annealing, and shape

History of This Topic

1930s-1980s: Hysteresis and the Preisach picture established that systems made of many bistable elements can remember turning points of past driving.
2004: Granular shear experiments showed a striking memory of direction, making it clear that even disordered grains can store a clean readout of past forcing.
2010s: Amorphous solids under cyclic shear revealed memories of amplitude, reversible plasticity, and a close connection between local rearrangements and hysterons.
2019: The broader field was synthesized in the review Memory formation in matter, helping define memory as a general nonequilibrium concept across materials.
2020s: The subject expanded into crumpled sheets, polycrystals, gels, and designed metamaterials, where coupling between memory units enables richer behavior and simple computation.
This review: Paulsen and Keim organize these developments into one teaching-friendly story, moving from disorder in amorphous solids toward purposeful design in mechanical materials.

What to Calculate First

  • Readout curveDefinition: A measured response during a controlled probe after training, such as rearrangements or torque versus readout amplitude. Why first: It is the clearest way to show that a memory is present and where it is stored.: Students should first learn how a memory is detected before worrying about deeper models.
  • Fraction of rearranging regionsDefinition: The amount of localized plastic activity triggered during readout. Why first: It links the visible memory signal to the underlying particle-scale events.: This is the cleanest route from microscopy or simulation data to physical interpretation.
  • Switching thresholds γ+ and γ-Definition: Strain values at which a hysteretic unit changes state on forward and reverse loading. Why first: These thresholds define hysterons and explain return-point memory.: They help students convert qualitative hysteresis loops into a simple model.
  • Memory capacityDefinition: The number of distinct past histories that can be distinguished by readout. Why first: It is the most direct way to compare direction memory, return-point memory, and designed computation.: This makes the lecture more than a catalog of examples; it gives students a unifying metric.

Main Teaching Points

  • Mechanical memory is a readable consequence of nonequilibrium history, not just vague path dependence.
  • In amorphous solids, localized rearrangements provide a mesoscopic language that explains why memory can be both robust and limited.
  • Annealing and short-term hysteretic writing are different mechanisms, and students should not confuse them.
  • Disorder is not only an obstacle; it can create the landscape that makes rich memory behavior possible.
  • Mechanical metamaterials turn these ideas from observation into design, making memory a route toward physical computing.

Four Memory Types to Compare

Memory of direction

Stored information: Whether the last shear was clockwise or counterclockwise.

Teaching value: A very intuitive first example because students can picture the write and read steps clearly.

Memory of amplitude

Stored information: The strain amplitude used during previous training.

Teaching value: It shows that a disordered solid can remember more than one binary feature of history.

Return-point memory

Stored information: Nested turning points of previous driving.

Teaching value: This is the cleanest place to introduce hysterons and ordered multiple-memory storage.

Elastic shape memory

Stored information: A preferred global shape encoded through geometry and local bistability.

Teaching value: It connects field-driven memory ideas to metamaterials and design.

Concept Glossary (Quick Lookup)

Mechanical memory

Definition: A material response that reveals information about past forcing or preparation.

Why important: It is the umbrella concept for the whole lecture.

Reversible plasticity

Definition: Plastic rearrangements that occur during loading but are undone when the loading is reversed.

Why important: It explains how apparently irreversible local motion can still support stable cyclic memories.

Hysteron

Definition: A two-state element with distinct forward and reverse switching thresholds.

Why important: It is the simplest model unit for many memory effects discussed in the review.

Return-point memory

Definition: A memory rule in which returning to a previous turning point restores the same state.

Why important: It gives an elegant explanation for nested multiple memories.

Annealing

Definition: Gradual relaxation under repeated driving toward a more stable or repeatable state.

Why important: It shows that memory can emerge through training over many cycles.

Mechanical metamaterial

Definition: A structured material whose mechanical response is set mainly by geometry and architecture.

Why important: It turns memory from an observed phenomenon into a design target.

Discussion Prompts

  • Which is more fundamental here: memory as a property of matter, or memory as a protocol-dependent readout?
  • Why does a review built around amorphous solids still spend so much time on crumpled sheets and metamaterials?
  • When does a hysteron model clarify the physics, and when does it hide too much microscopic detail?
  • If a material can be trained to remember, should we think of that as computation, adaptation, or only history dependence?

After-Class Assignment (Short)

  1. Choose one memory type from the lecture and explain its write protocol, readout protocol, and microscopic mechanism in one page.
  2. Sketch a simple hysteron model and use it to explain why nested amplitudes can or cannot be remembered.
  3. Propose one soft-matter or biophysical system where training by cyclic forcing might create a useful memory effect.
